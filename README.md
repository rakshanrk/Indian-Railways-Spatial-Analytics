# Indian-Railways-Spatial-Analytics

## Overview
A comprehensive Geospatial Analytics project using **MongoDB Atlas**, **Python**, and **Folium** to analyze the Indian Railways network.

## Key Features (15+ Operations)
- **Spatial Queries**: $near, $geoWithin, and $geoIntersects operations.
- **Advanced Analytics**: 3D Proximity modeling (Ooty), Centroid calculations, and Route LineStrings.
- **Data Source**: Indian Railways Dataset (Kaggle).
- **Stack**: MongoDB Atlas (2dsphere indexing), Geopy, Shapely, and Folium for mapping.
